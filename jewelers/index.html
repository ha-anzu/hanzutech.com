Collection Purpose

Built From Everyday Shop Friction

This section exists to collect tools that solve the repetitive problems hiding inside jewelry work: quoting, costing, design notes, admin cleanup, and small operational headaches that waste real time every day.

Open Source

Free For The Trade To Use

The plan is to keep these tools open source and free so any jeweler or designer can use them. Hanzu Tech also relies on them directly, which keeps the direction practical instead of hypothetical.

First App

Calculator Leads The Stack

Calculator is the first live app under Jewelers Tools. It gives jewelers a private quote-building workflow without storing client data, and it sets the standard for how the rest of the collection should feel.

Admin

Tools That Guard Margin

Directive

Start with the tools that fix everyday jewelry work, release them free, and keep refining them through real use.

Let the stack grow from real bench problems instead of bloated software ideas nobody actually needs.
