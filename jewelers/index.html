Purpose

Built From Everyday Shop Friction

This collection exists to turn repetitive quoting, costing, design admin, and production cleanup into tools that remove drag instead of adding software overhead.

Open Source

Free To Use, Refined In Production

These releases stay open and practical. Hanzu Tech uses them directly, so the direction keeps getting sharpened by actual work instead of speculative feature lists.

First Release

Calculator Leads The Stack

Calculator is the first live tool in Open Tools. It gives jewelers a private quote-building workflow without storing client data and sets the baseline for the rest of the collection.

Design

Tools That Respect Designers

These apps should help with messy references, dimensions, material logic, revisions, and presentation work without slowing the creative side down.

Admin

Tools That Guard Margin

The target is less repeated typing, less guesswork, fewer avoidable mistakes, and tighter operational control where money usually leaks out.

Pandora Box

Some tools will be niche or strange. That is fine. If they solve a real problem for a jeweler or designer, they belong in the collection.

Directive

Start with the tools that fix everyday jewelry work, release them free, and keep refining them through real use.

Let the stack grow from real bench problems instead of bloated software ideas nobody actually needs.
